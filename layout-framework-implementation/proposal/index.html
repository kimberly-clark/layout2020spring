Website using Bootsrap- OMGAA Special Olympics Webite

Proposal by Kimberly Clark

I want to create a OMGAA Special Olympics website, which is currenly just one page of the larger organizations website. It is outdated and does not have user friendly organization or a registration page.

OMGAA Special Olympics Website

Selected Framework: Bootstrap

Alerts- will be used to notify the user of upcoming deadlines or events.

Cards and Carousels- will be used to display information on the different sports offered.

Forms- will be used to create a registration page.

Navs- will be used to add necessary organization to the website.
